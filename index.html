ℹ How it works
Signing — Petra wallet signs your message using your private key via Ed25519. The wallet prefixes the message with APTOS\nmessage: ...\nnonce: ... before signing, producing a deterministic fullMessage and signature.

Verification — The Ed25519 signature is verified client-side using TweetNaCl. No data leaves your browser. The fullMessage + publicKey + signature are all you need to cryptographically prove ownership of an Aptos address.
